## 1. 개요
White Survivor는 2021년 출시한 뱀파이어 서바이버즈의 영향을 받아 만든 탑뷰형식의 로그라이크 슈터 게임입니다.</br>
구글 플레이 스토어에 개시하기 위한 테스트, 정책 등을 모두 경험해볼 수 있었습니다.</br>

## 2. 개발 환경
- Unity 2021.3.39f1 LTS
- C#
- Firebase
